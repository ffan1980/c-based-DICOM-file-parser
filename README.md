C based program to parse standard DICOM file to export all tags without using any third-party library.

how to compile: 
1. install xcode
2. open a command line to the program folder and type ./build and the executable file "dr" will be created

how to parse a DICOM file (e.g. xxx.dcm):
1. open a command line to the executable file "dr" folder and type ./dr xxx.dcm

This is writen for Mac (using o.m). If you are working on a Windows/Linux system, you need to rewrite the o.m file for the console outputs.
